Roots – Connect with your Heritage
Roots is a private, family-focused social media application designed to bridge the gap between generations. Unlike traditional social platforms, Roots focuses on secure family interactions, heritage preservation, and sharing memories within a closed circle.

🚀 Features
Family Networking: A dedicated space for private family interactions.

Heritage Sharing: Tools to document and share family history and "roots".

Media Gallery: Securely upload and store family-focused photos and memories.

Intuitive Navigation: Features a Bottom Navigation Manager for seamless switching between core app functions.

Family Management: Includes an Add Family Activity to easily expand your digital family circle.

🛠️ Tech Stack
Language: Java.

Platform: Android Studio.

Architecture: Model-View-Controller (MVC).

Build System: Gradle.

📁 Project Structure
The project is organized for scalability and maintainability:

AddFamilyActivity.java: Handles logic for adding new family members.

BottomNavManager.java: Manages the main navigation flow of the application.

res/layout: Contains the XML designs for a user-friendly interface.

AndroidManifest.xml: Defines app permissions and essential components.

🔧 Installation & Setup
Clone the repository: git clone https://github.com/your-username/Roots.git.

Open the project in Android Studio.

Ensure you have the latest Gradle wrapper installed.

Sync the project and run it on an emulator or physical Android device.
